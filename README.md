## Movies Info
The movies database - Using React, React Router and Redux

## Running on local machine
1. Clone the repository
  git clone https://github.com/bhojport/movies-info.git
2. Install node modules
  - Using NPM:
  npm install
  - Using yarn:
  yarn install
3. Start the project
  - Using NPM:
  npm start
  - Using yarn
  yarn start
